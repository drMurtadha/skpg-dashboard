# Dashboard SKPG-MyMoheS

Dashboard interaktif untuk analisis data lulusan sistem SKPG-MyMoheS.

## Features
- Analisis profil lulusan
- Status pekerjaan graduan
- Analisis pendapatan
- Visualisasi data interaktif

## Teknologi
- Python
- Streamlit
- Plotly
- Pandas
